Self-Improvement

The capability-scout watcher reads AgentIT's own fleet usage/effectiveness data and its own documented gaps, and proposes at most one small, evidence-cited change to AgentIT itself per cycle — landing as a draft PR for human review, never a direct commit. See the design doc for the full loop. GitHub's PR UI is where you review the actual code diff; this page is where you see what the loop considered and why, without needing to already know a PR exists.
